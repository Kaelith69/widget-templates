## 🧭 Quick Orientation

**widget-templates** is a repository of 20 animated SVG widgets, purpose-built for GitHub profile READMEs and personal websites. Each widget is a single `.svg` file with inline CSS animations — no external dependencies, no JavaScript, no build pipeline.

**The mental model:**
1. SVG is basically just XML
2. CSS animations work inside SVG
3. GitHub renders SVG files in markdown image tags
4. Therefore: animated graphics in your profile README, zero effort

**Who it's for:**
- Developers who want a GitHub profile that doesn't look like a bare `git init`
- People who enjoy the aesthetic of terminal UIs, VS Code, Finder, Spotify, etc.
- Anyone who has ever thought "I wonder if I could make my README look cooler"

## 🗂️ Widget Index

### Terminal Widgets (dark theme)
- `existential.svg` — matrix falling text + philosophical crisis
- `github-stats.svg` — mock GitHub statistics in a terminal window
- `joke.svg` — programming joke in a terminal window
- `rain.svg` — ASCII rain animation
- `skills.svg` — skill progress bars ("skill issue" chart)
- `status.svg` — current status/activity display

### Code & File Widgets
- `code-editor.svg` — VS Code-style Python editor with blinking cursor
- `file-explorer.svg` — macOS Finder-style file browser (light theme)

### Git Widgets
- `commit-history.svg` — GitHub-style commit history with branch graph
- `contribution-graph.svg` — contribution activity grid

### Utility Widgets
- `weather.svg` — animated sun, clouds, temperature
- `music-player.svg` — Spotify-style player with equalizer animation
- `calendar.svg` — monthly calendar (today highlighted)
- `system-monitor.svg` — CPU, RAM, disk, network gauges
- `time-zones.svg` — world clock, multiple zones
- `network-activity.svg` — network topology with animated packets

### Social & Progress Widgets
- `social-stats.svg` — follower counts and growth trends
- `progress-tracker.svg` — project progress bars with animated checkmarks
- `inspiration-quote.svg` — quote display with sparkles and gradient
