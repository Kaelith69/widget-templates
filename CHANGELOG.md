## [1.0.0] - 2026-02-24

### 🎉 Initial Release

Twenty widgets. Zero regrets (probably).

#### Added
- `existential.svg` — Matrix-style falling text animation. Why are we here. Why is any widget here.
- `github-stats.svg` — Terminal window with mock GitHub statistics. Looks very official.
- `joke.svg` — Terminal window with a programming joke. At least one of them is funny.
- `rain.svg` — ASCII rain in a terminal window. Genuinely calming. 10/10.
- `skills.svg` — "Skill issue" progress bar chart. Brutally honest.
- `status.svg` — Current status/activity terminal widget. Customize to taste.
- `code-editor.svg` — VS Code-style editor with syntax-highlighted Python and a blinking cursor. You look productive.
- `file-explorer.svg` — macOS Finder-style file explorer with light theme and sliding animations.
- `commit-history.svg` — GitHub-style commit history graph with branch visualization.
- `contribution-graph.svg` — GitHub-style contribution graph. All those green squares. Impressive.
- `weather.svg` — Animated weather widget with sun, clouds, and weather info.
- `music-player.svg` — Spotify-style music player with animated equalizer bars and playback controls.
- `calendar.svg` — Monthly calendar widget with today's date highlighted.
- `system-monitor.svg` — System monitor showing CPU, RAM, disk, and network usage.
- `time-zones.svg` — World clock displaying time in multiple time zones.
- `network-activity.svg` — Network topology visualization with animated data packets flying around.
- `social-stats.svg` — Social media statistics widget with follower counts and trends.
- `progress-tracker.svg` — Project progress bars with animated completion checkmarks.
- `inspiration-quote.svg` — Inspirational quote widget with sparkles and gradient. Believe in yourself, etc.
- `example-profile-README.md` — A complete example showing how to combine widgets in a real profile README.
- `README.md` — Documentation for humans.

## How to Read This

- **Added** — new widgets or features that didn't exist before
- **Changed** — modifications to existing widgets (colors, animations, layout)
- **Fixed** — something was broken and now it isn't
- **Removed** — something is gone (we'll try not to do this without warning)
- **Deprecated** — something will be gone soon (we'll tell you in advance)
